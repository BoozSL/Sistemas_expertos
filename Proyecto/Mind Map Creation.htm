Loan Risk Assessment System

Expert system for evaluating loan default risk

1. Database Loading

CSV data import & processing

  • Load train.csv file
  • Parse numeric columns
  • Handle missing values with coerce
  • Columns: Loan Amount, Interest Rate, DTI, etc.

2. Tooltip Component

Interactive help system

  • Custom ToolTip class
  • Show on mouse enter
  • Hide on mouse leave
  • Context-sensitive help

3. Expert System Rules

Risk evaluation logic

  • Risk Score Calculation:
  • • DTI > average: +1 point
  • • Inquiries > 3: +1 point
  • • Delinquency > 0: +2 points
  • • Interest > average: +1 point
  • • Accounts Delinquent > 0: +2 points
  • Classification:
  • • Score ≥ 4: HIGH RISK
  • • Score ≥ 2: MEDIUM RISK
  • • Score < 2: LOW RISK

4. User Interface

Bilingual Tkinter GUI

Input Fields
7 data entry fields
  • Loan Amount
  • Interest Rate (%)

  • Delinquency - two years
  • Inquiries - six months
  • Revolving Balance
  • Accounts Delinquent
Translation System
English/Spanish support
